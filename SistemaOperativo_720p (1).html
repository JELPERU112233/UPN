▶ USUARIO · SHELL · HARDWARE

El usuario, a traves del Shell, interactua
con el Hardware enviando comandos al kernel.

user@os:~$ _

▶ KERNEL · NUCLEO DEL SISTEMA

El Kernel es el nucleo del sistema operativo.
Brinda las caracteristicas esenciales al S.O. y se
comunica directamente con el Hardware en binario.

Scheduler  ●  Memory  ●  Drivers  ●  FS/IO
